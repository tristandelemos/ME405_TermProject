Detailed Description

This file contains the state machine which operates the turret.

The file requires several standard libraries as well as 5 custom made modules which control the encoders, the motors, the servo, and the camera. As well as a PID controller

Author
T DeLemos Created finite state machine
R Verleur Created servo driver, PID loop, and added byte operations to camera module
T Spicer Added functions to extract centroid/angle data from byte array

Function Documentation

◆ main()

def main.main ( )

The finite state machine for the turret operation.

This function steps through each state in turn and implements the appropriate transition logic. The program does not use cotask because the program was not able to allocate memory appropriately when cotask was included.

◆ shoot()

def main.shoot ( )

Fires the turret by moving the servo to set positions.

Fires the turret by moving the servo to a fire position, delaying then returning to a non firing position. Makes use of the custom servo module
