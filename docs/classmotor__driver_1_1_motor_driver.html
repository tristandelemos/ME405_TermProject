Detailed Description

This class implements a motor driver for an ME405 kit.

The class sets up a motor driver using 4 inputs: a forward and reverse pin, an enable pin, and a timer the motor driver should be used on the pins hooked up to the motor shield or else functionality is not likely

Creates a motor driver by initializing GPIO pins and turning off the motor for safety.

Uses the pins hooked up to the motor shield to create a motor driver with a PWM signal. The driver writes the enable pin to high but sets the duty cycle to 0 by default to prevent uncontrolled motor behavior

Parameters
en_pinA pin on the Nucleo corresponding to the enable pin for the current motor driver this should either be PA10 or PC1
in1pinThe pin corresponding to channel 1 of the motor driver should either be PB4 or PA0
in2pinThe pin corresponding to channel 2 of the motor driver should either be PB5 or PA1
timerAn integer corresponding to the timer attached to the motor driver should either be 3 or 5

Sets the duty cycle for the motor.

This method sets the duty cycle to be sent to the motor to the given level. Positive values cause torque in one direction, negative values in the opposite direction.

Parameters
