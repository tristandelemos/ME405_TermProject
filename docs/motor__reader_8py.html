Detailed Description

This file sends the required setpoint and gain to the nucleo over a serial communication.

The sent data works with motor_controller.py to control the motor using the pro_control.py file.

def motor_reader.main ( )

This is the main function which passes data to the micropython device.

This function uses the pyserial module to communicate over the serial port to the micropython. The function sends a setpoint and gain to the Nucleo and waits for the return data which represents the motor response. The data is then plotted and displayed.
